图例 · 数据契约

  • depth 0 / 无徽标 = 主 agent 行（parent_tool_use_id = NULL，渲染不变）
  • [子代理:<subagent_type>] 徽标 = 子代理产出行（parent_tool_use_id 指向主 agent 的 Task tool_use）
  • 缩进 / 深度数字 = depth 列（daemon session 级维护：主=0，子=父+1，嵌套子代理=2）
  • 同 parent_tool_use_id 连续行 视觉归组（嵌套模式可折叠，本期平铺模式仅缩进）
  • partial（流式 thinking/text）按 parent_tool_use_id 隔离 buffer，子代理长任务不白屏、不误清主 agent
