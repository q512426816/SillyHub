🟣 SillySpec ⚡ 技能 💻 命令行 📄 读文件 ✏️ 写文件 🔍 搜索 🤖 子任务 🌐 网搜 📋 清单 🔌 MCP ⚪ 其他
设计要点：第一层"工具"按钮仍控制是否显示所有工具调用；第二层在工具调用内部再细分具体工具。两层正交，可叠加（例：第一层点亮"工具"+第二层点亮"SillySpec/技能"=只看 SillySpec 和技能的调用）。
