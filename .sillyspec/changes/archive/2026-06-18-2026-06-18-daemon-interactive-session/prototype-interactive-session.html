现有 lease / WS / AgentRun / AgentRunLog / Redis / SSE / 权限审计全部保留。

2. AgentSession 状态机（D-003）

   create --> active -----------------------------> ended
                |   ^                               ^
                |   | idle (等下一条 inject)          | session.end / 空闲 30min (D-004)
                v   | running (turn 执行中)           |
              running --interrupt()--> 当前 AgentRun=failed, session 仍 active (spike D1)
                |
                | 进程崩溃 (Wave1/2)
                v
              failed   --Wave3--> reconnecting --resume(agent_session_id)--> active (spike D3)

Wave1/2 崩溃=failed（提示重开）；Wave3 resume 持久化。SDK 自动持久化 ~/.claude/projects/<cwd>/<sid>.jsonl。

3. turn -> AgentRun 映射（spike D4：result = 边界）

   inject msg1 -> [turn1: SDK assistant msgs...] -> result#1 -> 关闭 AgentRun#1
                                                            |
   inject msg2 -> [turn2: SDK assistant msgs...] -> result#2 -> 关闭 AgentRun#2
   (turn 级串行, 不支持运行中注入 -- spike S1)

   一个 AgentSession = N 个 AgentRun (run.agent_session_id FK)
   所有 turn 事件(含 Task 子agent)都在该 run 的 result 之前 -- 无孤儿事件

4. canUseTool 远程人审时序（决策2）

  SDK (claude 要调工具)
      | canUseTool(tool,input) 回调触发
      v
  ClaudeSdkDriver --WS: tool_approval_request--> backend --SSE/WS--> frontend
  (回调 await, claude 暂停)                                        | 弹审批卡
      ^                                                             | [Allow] [Deny]
      | resolve({allow|deny})                                       v
      +--------- allow/deny 回传 <-------- backend <--------- frontend
      |
      | 超时 5min 未响应 -> deny (默认)
      v
  SDK 继续执行 (allow) 或收到 deny.message  (spike D2: 回调可 await 任意延迟, 不超时)
