数据模型变更

workspaces (变更)

id UUID PK
name String(200)
slug String(100)
root_path String
status String(20)
tech_stack JSON NEW
build_command String NEW
test_command String NEW
component_key String(100) NEW
role String NEW
repo_url String NEW
default_branch String NEW
sillyspec_path REMOVED

workspace_relations (新表)

id UUID PK
source_id UUID FK → workspaces
target_id UUID FK → workspaces
relation_type String(50)
description String
created_at DateTime
UQ: (source_id, target_id, relation_type)
允许循环依赖 (A→B→A)
5 种关系类型: depends_on, consumes_api_from, tests, publishes_to, documents

change_workspaces / task_workspaces (新表)

change_id / task_id UUID FK
workspace_id UUID FK → workspaces
Change/Task 表保留原有字段
原 workspace_id 保留为"主 workspace"
新增 M:N 关联支持多 workspace

删除的表

project_components REMOVED
元数据合并到 workspaces 表
component_relations REMOVED
替换为 workspace_relations
