首次安装 daemon（新机器）—— 按操作系统切换

改动点：原折叠区块「首次安装 daemon」展开后，新增 「macOS / Linux ｜ Windows」 切换开关。 默认按浏览器 OS 自动选中（navigator.userAgent），可手动切换（应对远程部署：浏览器 OS ≠ daemon 目标机器 OS）。 Windows 给 PowerShell 一行 irm ... | iex；macOS/Linux 保持 curl ... | bash。

▼ 状态一：选中 Windows（PowerShell 一行）

▅首次安装 daemon（新机器）收起
▅irm http://127.0.0.1:3001/daemon/install.ps1 | iex

⚠️ 在 PowerShell 或 cmd 中运行（开始菜单搜「PowerShell」打开后粘贴）。

▼ 状态二：选中 macOS / Linux（curl|bash）

▅首次安装 daemon（新机器）收起
▅curl -fsSL http://127.0.0.1:3001/daemon/install.sh | bash -s -- --server-url http://127.0.0.1:3001

说明：上面两个 block 是同一个组件的两种 OS 选中态（实际只有一个，这里并排展示便于对比）。点切换按钮看交互。
