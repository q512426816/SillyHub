## Your Task

Analyze the change proposal and produce a structured brainstorm document:

1. **Requirements Clarification**: Identify ambiguities and ask clarifying questions.
2. **Solution Options**: Propose 2-3 solution approaches with trade-offs.
3. **Decision Record**: Recommend one approach with rationale.
4. **Scope Boundaries**: Define what is in-scope and out-of-scope.

Write the output to the change directory as question lists and decision records.

## Mode: WRITE

You have write access via worktree. Write brainstorm outputs (question lists, decision records) to the change directory.
